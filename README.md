# XemuHddImageEditor

This is a tool to edit the contents of a Xemu HDD Image. 

Right now it can read from and write to a raw HDD image, but I'm working on adding support for QCow2 Images.

I'm using this page for reference while building the QCow2 Image support:
[Qcow2 Image File Format](https://www.qemu.org/docs/master/interop/qcow2.html)

This is still in development and there will be bugs. Make a backup of your hdd image before making any changes in case something doesn't work right.
